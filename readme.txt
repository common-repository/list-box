== Description ==
List Box is a plugin show posts in [widgets wordpress](http://photoboxone.com/tag/widgets/).

The "List Box" widget.

### Settings:
1. Category: All, one category.
2. Displays: number posts.
3. Order (ID, Title, Description, ....) and (meta value)
4. Order by (DESC,ASC) 


View [documents wordpress](http://photoboxone.com/category/documents/).

== Installation ==
#### Thank you for your interest in List Box.
### Minimum requirements.  
*   Wordpress 3.0+  
*   PHP 5.x  
*   MySQL 5.x

This section describes how to install the plugin and get it working.
e.g.


1. Upload folder 'plugin-name' to the `/wp-content/plugins/` directory.
2. Activate the "List Box" plugin through the '[Plugins](http://photoboxone.com/category/plugins/)' menu in WordPress.
3. Go to [widgets wordpress](http://photoboxone.com/tag/widgets/). The "List Box" widget.

If any problem occurs, please read [documents wordpress](http://photoboxone.com/category/documents/).
